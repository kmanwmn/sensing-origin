# Git 手动提交流程

## 1. 查看当前文件改动状态

```bash
git status
```

会显示哪些文件被修改了（modified）或新增了（untracked）。

`index.html` 如果被修改了，会显示为红色 `modified: index.html`

---

## 2. 添加要提交的文件

添加 `index.html`：

```bash
git add index.html
```

如果想一次性添加所有改动文件：

```bash
git add .
```

---

## 3. 提交改动

```bash
git commit -m "这里写本次改动的说明"
```

示例：

```bash
git commit -m "修复中英文切换功能"
```

---

## 4. 推送到云端（GitHub）

```bash
git push
```

如果网络超时可以多试几次：

```bash
git push
```

---

## 完整示例

```bash
git status
git add index.html
git commit -m "修复中英文切换：补充页脚、规格表、团队、客户案例等板块的英文翻译"
git push
```
